4. De las funciones de forma a la rigidez K

La curvatura es: κ = d²w/dx² = B·d

Donde B = [N″₁, N″₂, N″₃, N″₄] es el vector de segundas derivadas.

El momento flector: M = E·I·κ = E·I·B·d

La energía de deformación (Principio de Trabajo Virtual):

U = ½·dT·(∫ E·I·BT·B dx)·d = ½·dT·K·d

Entonces K = ∫₀ᴸ E·I·Bᵀ·B dx

Ejemplo: K₂₂ = ∫ E·I·(N″₂)² dx

N″₂(ξ) = -(2/L_e)·(2 - 3ξ)

[N″₂]² = (4/L_e²)·(2 - 3ξ)² = (4/L_e²)·(4 - 12ξ + 9ξ²)

∫₀¹ [N″₂]² dξ = (4/L_e²)·[4ξ - 6ξ² + 3ξ³]₀¹ = (4/L_e²)·(4 - 6 + 3) = 4/L_e²

K₂₂ = E·I·L_e·(4/L_e²) = 4·E·I/L_e

K22 = 4 · E · ILe = 4 · 200000000 · 0.00013 = 26666.66667

11. Resumen

1. Discretizar → dividir en elementos

2. Funciones de forma N(ξ) → aproximar w(x) como polinomio

3. Derivar B = d²N/dx² → curvatura

4. Integrar K = ∫ EI·BᵀB dx → rigidez (Trabajo Virtual)

5. Ensamblar → sumar elementos en K global

6. Resolver K·d = F → desplazamientos

7. Post-proceso M = EI·B·d → esfuerzos
