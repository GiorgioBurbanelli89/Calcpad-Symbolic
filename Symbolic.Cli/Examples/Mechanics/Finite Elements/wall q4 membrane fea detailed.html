2. Funciones de forma bilineales Q4

El elemento Q4 tiene 4 nodos en coordenadas naturales ξ, η ∈ [-1, 1]:

Nodo 1: (-1,-1) esquina inferior izquierda

Nodo 2: (+1,-1) esquina inferior derecha

Nodo 3: (+1,+1) esquina superior derecha

Nodo 4: (-1,+1) esquina superior izquierda

Funciones de forma:

N1 ( ξ; η )  =  ( 1 − ξ )  ·  ( 1 − η ) 4

N2 ( ξ; η )  =  ( 1 + ξ )  ·  ( 1 − η ) 4

N3 ( ξ; η )  =  ( 1 + ξ )  ·  ( 1 + η ) 4

N4 ( ξ; η )  =  ( 1 − ξ )  ·  ( 1 + η ) 4

Propiedad de partición de la unidad: N₁+N₂+N₃+N₄ = 1 en cualquier punto

En el centro (0, 0):

N1  ( 0; 0 )  + N2  ( 0; 0 )  + N3  ( 0; 0 )  + N4  ( 0; 0 )  = 1

En un punto arbitrario (0.7, -0.4):

N1  ( 0.7; -0.4 )  + N2  ( 0.7; -0.4 )  + N3  ( 0.7; -0.4 )  + N4  ( 0.7; -0.4 )  = 1

Valores en los nodos: N_i vale 1 en su nodo y 0 en los demás

N₁ en nodo 1 (-1,-1):

N1  ( -1; -1 )  = 1

N₁ en nodo 2 (+1,-1):

N1  ( 1; -1 )  = 0

Derivadas respecto a ξ:

dN1dξ ( η )  = - ( 1 − η ) 4

dN2dξ ( η )  = 1 − η4

dN3dξ ( η )  = 1 + η4

dN4dξ ( η )  = - ( 1 + η ) 4

Derivadas respecto a η:

dN1dη ( ξ )  = - ( 1 − ξ ) 4

dN2dη ( ξ )  = - ( 1 + ξ ) 4

dN3dη ( ξ )  = 1 + ξ4

dN4dη ( ξ )  = 1 − ξ4

Verificación: Suma de derivadas = 0

dN1dξ  ( 0 )  + dN2dξ  ( 0 )  + dN3dξ  ( 0 )  + dN4dξ  ( 0 )  = 0

dN1dη  ( 0 )  + dN2dη  ( 0 )  + dN3dη  ( 0 )  + dN4dη  ( 0 )  = 0

5. Jacobiano

Para elementos rectangulares uniformes, el Jacobiano es constante:

J11 = dx2 = 0.6252 = 0.3125

J22 = dy2 = 0.52 = 0.25

detJ = J11 · J22 = 0.3125 · 0.25 = 0.078125

Jacobiano inverso:

J⁻¹₁₁ = 1/J₁₁ = Jinv11 = 1J11 = 10.3125 = 3.2

J⁻¹₂₂ = 1/J₂₂ = Jinv22 = 1J22 = 10.25 = 4

6. Matriz B (deformación-desplazamiento)

Para esfuerzo plano, las deformaciones son:

ε_xx = ∂u/∂x, ε_yy = ∂v/∂y, γ_xy = ∂u/∂y + ∂v/∂x

B_i = [∂N_i/∂x, 0 | 0, ∂N_i/∂y | ∂N_i/∂y, ∂N_i/∂x]
