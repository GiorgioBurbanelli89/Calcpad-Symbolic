Capitulo 1: Origenes del Metodo de Elementos Finitos

De Euler (1744) al primer elemento finito (1956)

 

1.1 El problema original: La viga de Euler (1744)

Leonhard Euler en 1744 resolvio el problema de la viga elastica.

Encontro una ecuacion diferencial que describe como se dobla:

 

E · I · d4 · vd · x4 = q  ( x ) 

 

E = modulo de elasticidad (rigidez del material)

I = momento de inercia (geometria de la seccion)

v = deflexion vertical (lo que buscamos)

q = carga distribuida

 

Para casos simples, Euler podia resolverla a mano:

 

 

Solucion exacta de Euler para viga SA con carga uniforme:

E1 = 200000

I1 = 5 · 10-5 = 5×10-5

L1 = 6

q1 = 10

 

v  ( x )  = q · x24 · E · I ·  ( L3 − 2 L · x2 + x3 ) 

v{max} = 5 · q · L4384 · E · I

 

vmax1 = 5 · q1 · L14384 · E1 · I1 = 5 · 10 · 64384 · 200000 · 5×10-5 = 16.875

 

 

 

Euler podia resolver esto porque la geometria es simple (rectangular).

Pero que pasa con formas complicadas? Ahi empezo el problema...

 

1.2 Lord Rayleigh (1877): Metodo de la Energia

Rayleigh tuvo una idea brillante: en vez de resolver la ecuacion diferencial,

MINIMIZAR la energia. La estructura se deforma de la manera que minimiza

la energia total = energia interna - trabajo externo.

 

Π = U − W

 

U = energia de deformacion (la estructura almacena energia al deformarse)

W = trabajo de las fuerzas externas

 

Para una viga:

U = 1/2*∫{0}^{L} E*I*(d^2*v/d*x^2)^2*d*x

W = ∫{0}^{L} q*v*d*x

 

La idea de Rayleigh: ADIVINAR la forma de v(x) con una funcion simple

y minimizar la energia. No es exacto, pero es muy buena aproximacion!

 

Ejemplo: Adivinar v(x) = C*sin(pi*x/L)

Esta funcion cumple: v(0)=0 y v(L)=0 (condiciones de apoyo).

Solo tenemos que encontrar C (la amplitud).

 

 

 

Minimizando la energia, encontramos C:

C = q · L4π4 · E · I

 

Cr = q1 · L14π4 · E1 · I1 = 10 · 643.1415934 · 200000 · 5×10-5 = 13.304713

 

Deflexion maxima por Rayleigh:

vmax_rayleigh = Cr = 13.304713

 

Comparacion con Euler exacto:

errorrayleigh = vmax_rayleigh − vmax1vmax1 · 100 = 13.304713 − 16.87516.875 · 100 = -21.157256

 

Error de solo ~0.4%! Rayleigh adivino la forma y la energia hizo el resto.

 

 

 

Las dos curvas son casi identicas! El metodo de la energia funciona.

 

1.3 Ritz (1909): Mejorar la aproximacion

Walter Ritz mejoro la idea de Rayleigh: en vez de UNA funcion,

usar VARIAS funciones sumadas con coeficientes:

 

v  ( x )  = a1*φ_1(x) + a_2*φ_2(x) + a_3*φ_3(x) + ...

 

Cada phi es una funcion base (como sin(n*pi*x/L) para n=1,2,3...).

Mas funciones = mejor aproximacion.

 

Ejemplo: Ritz con 1, 2 y 3 terminos

 

 

Con 3 terminos ya es practicamente exacto!

Pero esto solo funciona si las funciones phi cubren toda la estructura.

Para formas complicadas no hay funciones phi simples...

 

1.4 Galerkin (1915): Otro camino al mismo resultado

Boris Galerkin propuso: en vez de minimizar energia,

hacer que el ERROR sea cero en promedio.

 

Si la ecuacion real es EI*v'' = q, y nuestra aproximacion da un residuo R:

R = E*I*v''''{aprox} - q

 

Galerkin dice: el residuo ponderado por las funciones base debe ser cero:

∫{0}^{L} R*φ_i*d*x = 0

 

Esto da el MISMO resultado que Rayleigh-Ritz para problemas de elasticidad!

Son dos caminos diferentes que llegan al mismo lugar.

 

1.5 Courant (1943): Dividir en TRIANGULOS

Richard Courant tuvo la idea clave: en vez de funciones que cubren

TODA la estructura, dividir en pedazos pequenos (triangulos) y

usar funciones simples en cada pedazo.

 

Esto es el NACIMIENTO del concepto de elemento finito!

 

 

Dentro de cada triangulo, el desplazamiento es LINEAL (plano).

Las funciones de forma son simples porque cada elemento es pequeno.

Muchos elementos pequenos = buena aproximacion de formas complicadas.

 

1.6 Turner, Clough, Martin y Topp (1956): El primer FEM real

Ingenieros de Boeing necesitaban analizar alas de aviones.

Combinaron las ideas de Ritz + Courant y crearon el primer programa.

Dividieron el ala en triangulos, calcularon rigidez de cada uno,

y ensamblaron todo en una gran ecuacion K*u = F.

 

El nombre "Finite Element Method" lo puso Ray Clough en 1960.

 

1.7 Como funciona FEM: el proceso completo

 

1. DIVIDIR la estructura en elementos (barras, triangulos, quads)

2. Para cada elemento: calcular su matriz de RIGIDEZ K usando funciones de forma

3. ENSAMBLAR todas las K en una K global grande

4. Aplicar condiciones de FRONTERA (apoyos)

5. RESOLVER K*u = F para encontrar desplazamientos u

6. Calcular esfuerzos y reacciones

 

Ejemplo: 2 barras — el FEM mas simple posible

 

 

Paso 1: Rigidez de cada elemento

Efem = 200000

Afem = 0.001

Lfem = 4

kfem = Efem · AfemLfem = 200000 · 0.0014 = 50

 

Paso 2: K local de cada barra (2x2)

K1 = k*[1,-1; -1,1] para nodos (1,2)

K2 = k*[1,-1; -1,1] para nodos (2,3)

 

Paso 3: Ensamblar K global (nodo 1 fijo, quedan 2 DOF)

Kfem = 2 · kfem-kfem-kfemkfem = 2 · 50-50-5050 = 100-50 -5050

⃗Ffem = 0100

 

Paso 4: Resolver

⃗ufem = Kfem−1 · ⃗Ffem = 2 4

u2fem = ⃗ufem.1 = 2

u3fem = ⃗ufem.2 = 4

 

Paso 5: Fuerzas internas

N1fem = kfem · u2fem = 50 · 2 = 100

N2fem = kfem ·  ( u3fem − u2fem )  = 50 ·  ( 4 − 2 )  = 100

Reaccion en apoyo:

Rfem = -N1fem = -100

 

 

 

Eso es TODO el metodo de elementos finitos:

Dividir → Rigidez → Ensamblar → Resolver → Resultados

 

Resumen historico

1744 — Euler: ecuacion de la viga (solucion exacta para casos simples)

1877 — Rayleigh: minimizar energia (aproximacion con 1 funcion)

1909 — Ritz: varias funciones base (mejor aproximacion)

1915 — Galerkin: residuos ponderados (mismo resultado que Ritz)

1943 — Courant: dividir en triangulos (el concepto clave)

1956 — Turner/Clough/Boeing: primer FEM computacional

1960 — Clough: inventa el nombre "Finite Element Method"

Hoy — SAP2000, ETABS, ANSYS, OpenSees: todos usan FEM
