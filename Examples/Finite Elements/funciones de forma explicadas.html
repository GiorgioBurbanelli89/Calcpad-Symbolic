Funciones de Forma — Explicacion paso a paso

 

Que problema queremos resolver?

Conocemos el desplazamiento en los EXTREMOS de una barra.

Pero queremos saber el desplazamiento en CUALQUIER punto intermedio.

 

 

La respuesta: INTERPOLACION

Si conocemos u1=0 en un extremo y u2=0.5 en el otro,

el desplazamiento en cualquier punto x es una mezcla:

 

u  ( x )  = N1  ( x )  · u1 + N2  ( x )  · u2

 

N1 y N2 son las FUNCIONES DE FORMA. Son "pesos" que dicen

cuanto influye cada nodo en el punto x.

 

Como funcionan N1 y N2?

 

Imaginalo asi: estas parado en un punto x de la barra.

Si estas CERCA del nodo 1, el desplazamiento se parece a u1.

Si estas CERCA del nodo 2, se parece a u2.

N1(x) te dice "cuanto pesa el nodo 1" y N2(x) "cuanto pesa el nodo 2".

 

Las formulas son simples:

N1  ( x )  = 1 − xL

N2  ( x )  = xL

 

Veamos que pasa en puntos clave:

L = 10

 

En x=0 (nodo 1): N1=1, N2=0 → u = 1*u1 + 0*u2 = u1 ✓

N1en_0 = 1 − 0L = 1 − 010 = 1

N2en_0 = 0L = 010 = 0

 

En x=L (nodo 2): N1=0, N2=1 → u = 0*u1 + 1*u2 = u2 ✓

N1en_L = 1 − LL = 1 − 1010 = 0

N2en_L = LL = 1010 = 1

 

En x=L/2 (mitad): N1=0.5, N2=0.5 → u = 0.5*u1 + 0.5*u2 (promedio!)

N1en_mitad = 1 − L2L = 1 − 10210 = 0.5

N2en_mitad = L2L = 10210 = 0.5

 

Grafica de N1 y N2:

 

 

Observa: cuando N1 sube N2 baja. Siempre suman 1 (N1+N2=1).

Esto significa: en cualquier punto el desplazamiento es un promedio ponderado.

 

Ejemplo numerico completo

 

Barra con u1=0 (fijo) y u2=0.5 (se mueve por la fuerza P):

 

 

u1 = 0

u2 = 0.5

 

El desplazamiento en x=3:

xtest = 3

N1test = 1 − xtestL = 1 − 310 = 0.7

N2test = xtestL = 310 = 0.3

uen_3 = N1test · u1 + N2test · u2 = 0.7 · 0 + 0.3 · 0.5 = 0.15

 

El desplazamiento en x=7:

xtest2 = 7

N1test2 = 1 − xtest2L = 1 − 710 = 0.3

N2test2 = xtest2L = 710 = 0.7

uen_7 = N1test2 · u1 + N2test2 · u2 = 0.3 · 0 + 0.7 · 0.5 = 0.35

 

Desplazamiento a lo largo de toda la barra:

 

 

Es una linea recta! Porque N1 y N2 son lineales (rectas).

Para una barra esto es EXACTO. La barra se estira uniformemente.

 

Y si la barra se DOBLA? Necesitamos curvas!

Un resorte o barra solo se estira. Pero una VIGA se dobla.

Una linea recta no puede describir una curva de flexion.

Necesitamos funciones de forma CUBICAS (polinomios de grado 3).

 

 

Cada nodo de la viga tiene DOS valores:

- v = desplazamiento vertical (cuanto sube/baja)

- theta = rotacion (cuanto gira)

Entonces necesitamos 4 funciones de forma (2 nodos x 2 valores):

 

v  ( x )  = N1 · v1 + N2 · θ1 + N3 · v2 + N4 · θ2

 

N1: efecto de v1 (desplazamiento en nodo 1)

N2: efecto de theta1 (rotacion en nodo 1)

N3: efecto de v2 (desplazamiento en nodo 2)

N4: efecto de theta2 (rotacion en nodo 2)

 

Las 4 funciones de Hermite:

 

 

 

N1 y N3 son como las funciones lineales pero CURVAS (cubicas).

N2 y N4 son nuevas: representan el efecto de la ROTACION.

Son mas pequenas porque la rotacion tiene menos efecto que el desplazamiento.

 

Ejemplo: Voladizo con P=10 kN

 

 

Nodo 1 empotrado: v1=0 y theta1=0 (no se mueve ni gira)

Nodo 2 libre: v2=? y theta2=? (lo que buscamos)

Entonces u(x) = N3*v2 + N4*theta2 (porque v1=0 y theta1=0)

 

Ev = 200000

Iv = 8.333 · 10-6 = 8.333×10-6

Lv = 3

Pv = 10

 

Solucion analitica:

v2 = -Pv · Lv33 · Ev · Iv = -10 · 333 · 200000 · 8.333×10-6 = -54.00216

θ2 = Pv · Lv22 · Ev · Iv = 10 · 322 · 200000 · 8.333×10-6 = 27.00108

 

Deflexion: v(x) = N3*v2 + N4*theta2*L

 

 

La curva verde es la SUMA de la azul (efecto desplazamiento) + roja (efecto rotacion).

Eso es una funcion de forma: interpola la solucion a partir de los valores nodales.

 

Resumen

1. Las funciones de forma son PESOS que dicen cuanto influye cada nodo.

2. Para barras son LINEALES (rectas): N1=1-x/L, N2=x/L.

3. Para vigas son CUBICAS (curvas): Hermite N1,N2,N3,N4.

4. El desplazamiento en cualquier punto = suma de N*valor_nodal.

5. De las funciones de forma se obtiene la RIGIDEZ integrando sus derivadas.
